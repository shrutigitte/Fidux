# DesignFlow MVP Artifacts

This folder contains the finalized MVP contract for DesignFlow.

- `designflow-e2e-free-first-v1.md`: complete end-to-end architecture and implementation plan with free-tier service strategy and API key checklist.
- `plugin-runbook.md`: operational runbook to start backend, seed plugin PAT, and run end-to-end smoke checks.
- `mvp-decisions.md`: locked policy decisions.
- `rbac-policy.md`: route-level authorization matrix.
- `websocket-contract.md`: room model, auth, and event payloads.
- `openapi/designflow-mvp.yaml`: API contract for plugin and core web flows.
- `../../prisma/schema.prisma`: database model and indexes.
